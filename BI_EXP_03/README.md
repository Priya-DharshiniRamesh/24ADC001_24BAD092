# Business Intelligence Lab – Experiment 3
## Data Cleansing using Tableau & Power BI

This repository contains the implementation of **Business Intelligence Experiment 3**, where Tableau and Power BI were used to identify and handle missing values, outliers, and inconsistencies in a sales dataset. The experiment demonstrates various data cleansing techniques and validates the cleaned data through interactive dashboards.

---

## 🎯 Objective

To clean datasets by identifying and handling missing values, outliers, and inconsistencies using the built-in features of **Tableau** and **Power BI**, ensuring the data is accurate, consistent, and ready for analysis.

---

## 🛠️ Tools Used

- Tableau Desktop (Sample - Superstore Dataset)
- Microsoft Power BI Desktop (Financials / Sales Dataset)
- Power Query
- Calculated Fields
- DAX Measures

## 📈 Key Findings

- The dataset contained **no missing values** in the Sales, Profit, and Region fields.
- Data completeness was verified as **100%**.
- Region values were already standardized as **Central, East, South, and West**.
- Outliers were identified but retained because they represented valid business transactions.
- Scatter plots and box plots effectively highlighted extreme values.
- Summary metrics confirmed that the dataset was clean and ready for analysis.
